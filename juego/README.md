# 🎭 El Impostor - Juego en Python

Un juego tipo "Among Us" programado en Python con interfaz gráfica en Tkinter. ¡Perfecto para jugar en clase!

## 📋 Descripción

**El Impostor** es un juego local donde:
- Uno de los jugadores es elegido al azar como **impostor**
- El resto de jugadores son **civiles** y conocen una **palabra secreta**
- Los civiles deben adivinar quién es el impostor basándose en las palabras que dicen
- Al final, **todos votan** para expulsar al que crean que es el impostor
- **¡Ganan los civiles si expulsan al impostor, o gana el impostor si logra pasar desapercibido!**

## 🎮 Características

✅ Máximo de jugadores configurable  
✅ Asignación aleatoria de roles  
✅ Palabra secreta personalizada  
✅ Sistema de votación por mayoría  
✅ Revelación de roles al final  
✅ Opción para jugar nuevas partidas  
✅ Interfaz gráfica simple con Tkinter
